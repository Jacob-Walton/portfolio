## 소개

이 웹사이트는 저 자신과 이전에 완성한 프로젝트에 대한 자세한 정보를 제공하기 위해 사용하는 개인 포트폴리오입니다. 이 웹사이트는 Next.js 14를 사용하여 구축되었습니다.

## 사용된 기술

- Next.js 14
- TypeScript
- CSS 모듈
- Framer Motion
